📂 File Requirements

  • First column must be named exactly Timestamp.
  • Sensor file must include T and RH columns.
  • Selected parameter columns must match exactly (case-sensitive).
  • Accepts: .csv · .xls · .xlsx.

⚙️ Assumptions

  • Timezone differences are ignored — raw parsed timestamps used.
  • Resampling uses interval-end labeling aligned to local timezone.
  • Merge uses nearest-neighbor with auto-detected reference interval tolerance.
  • MLR uses normal equations in pure JavaScript (no external library).
  • Timestamp export format: DD-MM-YYYY HH:mm:ss.
